Hotkeys
The most important hotkeys to know:
Ctrl + C - Copies underlined text
Ctrl + P - Pastes copied text
Ctrl + Tab - Switch between open programs
Ctrl + Alt + T - Open terminal windows
Ctrl + Shift + alt + del - Opens a task Manager to Force Stop Programs
