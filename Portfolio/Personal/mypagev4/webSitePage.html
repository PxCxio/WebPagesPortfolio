Syntax
HTML is defined by tags < tag> < /ctag>
Everything in between HTML tags are considered an html content and in a way you could consider this an html webpage. < html> < /html>
Most common tags inside html are the head tag, where you input all Informatiof the page on for indexing and of authorship; and the body tag, where you put all the content of the page
table tags, tr (tablerow) tags, td (tabledown[column]) tags. With these tags you can create a table with rows and columns and add width and height to give it structure.
CSS Cascading Style Sheets is another sheet which adds style/format to the specific tags, and tags that can have IDs or Classes that way all of those tags inherit the specific style/format that you have indicated in this sheet. Since CSS3 you can also add CSS grid and add dynamic formating straight from css!
