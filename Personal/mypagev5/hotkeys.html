MacOS

Command + Spacebar + type ≈ 3 letters of the Application/Program you want to open
Command + Tab to switch amongst open Applications [+ Shift to cycle inversly]
Command + Tilde ` to switch amongst the different windows of the open application [+ Shift to cycle inversly]
Control + Tilde ` to switch amongst the different tabs within one window [+ Shift to cycle inversly]
Command + n to Open a new Window of the same application
Command + w to close current highlighted window
Command + q to close current app
Control + F2 [+Alt, if your F2 Control Media like volume and brightness] to highlight the top menu where the MacOS Apple is located

Firefox/Chrome/Safari [Internet Browsers]


Command + t = Open a new tab
Command + Shift + t Re-open recently closed Tab

Windows

WindowsKey + type ≈ 3 letters of the Program/Application you want to open.
Alt + Tab to switch amongst open Program windows. [+ Shift to cycle inversly]
Control + Tab to switch amongst the tabs of the same applications [+ Shift to cycle inversly]
Alt + f then x or Alt + F4 or Control + w to close current window
Alt + f then n or Control + n to Open a new Window of the current Program you are using.
Control + alt + supreme/delete Bring task manager menu

Firefox/Chrome/Safari [Internet Browsers]

Linux

Alt + F2 + ≈ 3 letters of the Application you want to Run..
Control + Alt + t to open a Terminal Window
Alt + Tab to switch amongst open Programs.s [+ Shift to cycle inversly]
Alt + Tilde ` to switch amongst different open windows of the same highlighted applications [+ Shift to cycle inversly]
